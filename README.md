## 눈송 시네마에 오신 걸 환영합니다❄

Python과 MSSQL을 활용해 만든 영화 예매 프로그램입니다.<br><br>
해당 프로그램은 회원가입, 로그인, 현재 상영중인 영화 확인, 영화 예매, 예매 영화 조회, 예매 내역 삭제, 회원 정보 수정의 기능을 제공합니다.

기본적인 스키마는 이렇습니다.<br><br>
